STALZONE Auction Watcher - portable version

Запуск:
1. Перенесите всю папку целиком на компьютер с Windows 10/11.
2. Запустите STALZONE-Auction-Watcher-portable.exe.
3. Не перемещайте отдельно EXE: .env и файлы базы должны оставаться рядом с ним.

Файлы данных:
- .env - API-ключи и настройки уведомлений. Не передавайте этот файл посторонним.
- market_cache.sqlite3 - локальная история рынка и продаж.
- auction_watchlist.json - правила мониторинга.
- .auction_seen.json - уже просмотренные лоты.

Для работы нужен Microsoft Edge WebView2 Runtime. В актуальных Windows 10/11 он обычно уже установлен.
Перед копированием или резервным копированием закройте приложение.
